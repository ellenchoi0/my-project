# Development of a Traffic Light Controller Using Raspberry Pi

라즈베리파이와 Python을 활용하여 자동차 및 보행자용 신호등 시스템을 구축하고, 하드웨어와 소프트웨어 간의 상호작용을 제어하는 실습 프로젝트입니다.

> 본 프로젝트는 'gpiozero' 라이브러리를 통해 실제 신호등의 점등 시퀀스를 구현하며,
> 예외 처리(인터럽트)를 통한 프로세스 종료 및 자원 회수 메커니즘을 심층적으로 다룹니다.

## 개요

라즈베리파이의 GPIO 인터페이스를 통해 LED 회로를 구성하고, 시간 차 점등 로직을 구현합니다. 실행 환경(터미널 vs 프로그래밍 앱)에 따른 동작 특성을 비교 분석하고, 무한 루프 상태의 프로그램을 안전하게 종료하는 인터럽트 개념을 학습합니다.

## 핵심 특징

- **실시간 신호 제어**: 자동차(3색) 및 보행자(2색) 신호등의 논리적 연동 구현
- **안전한 회로 설계**: 오옴의 법칙(V = IR)에 근거한 전류 제한 저항 적용
- **인터럽트 처리**: KeyboardInterrupt를 통한 프로세스 강제 종료 시 LED 전원 완전 차단
- **객체 지향 제어**: gpiozero.LED 객체를 활용한 직관적인 하드웨어 추상화 제어

## 하드웨어 아키텍처

본 프로젝트는 [라즈베리파이의 BCM 핀 배열](https://www.raspberrypi.com/documentation/computers/raspberry-pi.html#gpio)을 기준으로 다음과 같이 설계되었습니다.

## 회로 구성 이론

- **LED 동작 원리**: 전기 에너지를 빛으로 변환하는 반도체 소자의 특성 및 극성 활용
- **저항 산출**: 3.3V 전압 환경에서 LED 보호를 위해 최소 130Ω 이상의 저항이 필요

## 실행 및 설치

### 1. 의존성 설치

본 프로젝트는 라즈베리파이 OS에 내장된 gpiozero 라이브러리를 사용합니다.

### 2. 서버 실행

터미널에서 소스코드가 있는 디렉토리로 이동 후 'python3 main2.py' 또는 'python3 main2-1.py' 명령어를 입력합니다.

### 3. 종료(Interrupt)
- 실행 중 Ctrl + C를 입력하면 KeyboardInterrupt가 발생하며 프로그램이 종료됩니다.
